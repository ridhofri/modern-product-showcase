## ğŸ¨ Customization

### Add Products

Edit `assets/data.json`:

```json
{
  "id": "product-1",
  "title": "Product Name",
  "subtitle": "Description",
  "image": "assets/images/product.jpg",
  "color": "Black",
  "battery": "20 Hours",
  "weight": "250g",
  "latency": "40ms",
  "price": "Rp 2.500.000"
```

In `style.css`:

```css
/* Primary color */
background: linear-gradient(135deg, #ff4500, #ff6347);
```
